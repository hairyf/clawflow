# Agent Skills

Index of skills available to agents in this repository.

### agent-flow

- **Location**: `skills/agent-flow/`
- **Description**: Flow-Agent multi-agent orchestration and multi-provider LLM runtime. Covers Swarm/Hive-Mind, MCP tools, Codex provider, SPARC methodology, and architecture reference. Based on [whrit/agent-flow](https://github.com/whrit/agent-flow).

### claude-flow

- **Location**: `skills/claude-flow/`
- **Description**: Claude-Flow v3 enterprise AI orchestration: multi-agent swarms, self-learning hooks, AgentDB/HNSW memory, MCP server, security, and RuVector intelligence. Based on [ruvnet/claude-flow](https://github.com/ruvnet/claude-flow).

### openclaw

- **Location**: `skills/openclaw/`
- **Description**: OpenClaw personal AI assistant platform. Gateway control plane, CLI workflows, tools (browser/canvas/nodes), sandboxing + security controls, model/auth failover, and remote access patterns. Based on [openclaw/openclaw](https://github.com/openclaw/openclaw).
